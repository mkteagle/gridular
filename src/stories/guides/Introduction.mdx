# Virtualized Grid

A high-performance, feature-rich data grid component for React with virtualization support.

## Overview

The Virtualized Grid is a comprehensive solution for displaying large datasets with optimal performance. It combines the power of TanStack Virtual for efficient rendering with a rich set of features for data manipulation and presentation.

## Performance Tips

1. **Use `useMemo`** for data generation to prevent re-renders
2. **Enable virtualization** for datasets > 50 rows
3. **Optimize render functions** - avoid creating new objects/functions on each render
4. **Use column memoization** - define columns outside component or with `useMemo`
5. **Consider pagination** for very large datasets (10,000+ rows)
